// how it works
01 —

Frame Capture

OpenCV captures live video frames from your webcam at high speed, feeding them into the analysis pipeline.

02 —

Face Detection

DeepFace locates faces in each frame using a robust detection model, identifying bounding box coordinates.

03 —

Emotion Analysis

The cropped face region is passed through a deep learning model that outputs probability scores for 7 emotions.

04 —

Flask API Response

Results are served via a Flask REST endpoint — this frontend polls it and renders emotion data in real time.
